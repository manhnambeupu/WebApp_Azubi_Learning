# GitHub Codespaces

Mo repo bang GitHub Codespaces. Sau khi Codespaces load xong, mo Terminal va chay:

```bash
docker-compose up -d
```

Vao tab **Ports**, click chuot phai vao cac cong `3000`, `3001`, `9000` va doi tu **Private** sang **Public**.

Sau do copy link cua cong `3000` de gui cho nguoi khac test giao dien web.
